# Hi there, I'm Mohamed Nouri AKA Ophanim

Full-Stack Developer | Security Researcher | Indie Game Dev

I’m a developer who lives at the intersection of high-performance backend systems and interactive user experiences. My approach to code is simple: build it to be fast, keep it modular, and always design for the long term.
Technical Craft

  Systems & Backend: I spend my time architecting performant applications using Go and Rust. I’m comfortable working close to the metal, managing memory, and implementing concurrency patterns that actually scale.

  Web Ecosystem: Whether it's Next.js or custom WebSocket implementations for real-time projects, I enjoy creating fluid, responsive web interfaces that don't choke under pressure.

  Data & Infrastructure: My toolkit centers on SQLite3—I’m a big fan of its versatility, especially when configured with WAL mode for performance. I use Docker to keep my development environments isolated, reproducible, and ready for deployment.

  Security Research: In my spare time, I’m a Bug Bounty Hunter and pentester. I don’t just build systems; I try to break them to understand how they can be made more resilient.

  Game Development: I build 2D indie games from the ground up. To keep things performant at a locked 60fps, I rely on ECS (Entity Component System) architectures rather than traditional class hierarchies. It’s cleaner, data-oriented, and handles complex game states far better.

  Design & Architecture: I bridge the gap between UI/UX design and technical implementation. When architecting systems, I gravitate toward proven design patterns—like using Singletons for global configuration managers or OO patterns where they actually provide structural value without adding bloat.
